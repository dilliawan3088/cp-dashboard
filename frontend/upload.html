    Format Requirements

    Your Excel file must contain exactly 9 columns in this exact order:

    1. NO - Row number
    2. TRUCK NO - Truck identifier
    3. D/O Number - Delivery order number
    4. FARM - Farm name
    5. D/O Quantity - Numeric value
    6. BIRD COUNTER - Numeric value
    7. TOTAL SLAUGHTER - Numeric value
    8. DOA - Numeric value (Dead on Arrival)
    9. NON HALAL - Numeric value

    Important:

    • Column headers must be in row 1
    • Data rows start from row 2
    • Columns 5-9 must contain numeric values only
    • No additional columns are allowed
    • At least one data row is required
